# create-histogram

This project uses the Titanic dataset to make simple graphs.
The graphs help us understand the passengers on the Titanic and who survived.

1. Files
File	What it contains
Titanic-Dataset.csv	Data of Titanic passengers
create-histogram-and-bar-chart.ipynb	Code (Jupyter Notebook) to make graphs
2. What the notebook does

The Jupyter Notebook:

Loads the Titanic data

Cleans the data

Makes a histogram of Age

Makes bar charts for:

Survived / Not Survived

Male / Female

Passenger Class (1st, 2nd, 3rd)

3. Tools used

Python

Pandas

Matplotlib

Jupyter Notebook

4. How to run

Install the required Python libraries:

pip install pandas matplotlib


Open the notebook:

jupyter notebook


Run the file:

create-histogram-and-bar-chart.ipynb

5. Output

You will see:

A histogram of Age

A bar chart of survival count

A bar chart of male vs female

A bar chart of passenger class

6. Purpose

To understand Titanic passengers better using simple graphs.
